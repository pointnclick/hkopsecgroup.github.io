## OPSEC (Operational Security) 嘅定義
行動安全(OPSEC) 係一門針對避免泄漏情報嘅學問。假如有情報泄漏咗畀敵方，好可能導致整個行動失敗，甚至影響當事人、同伴嘅安全。行動之前，需要分析整個行動所依賴嘅情報，訂立一層層嘅防禦手段以確保密性。
